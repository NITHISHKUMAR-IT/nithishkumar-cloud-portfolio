# Nithishkumar K — Cloud & DevOps Portfolio

A custom, Apple-inspired corporate portfolio website built with semantic HTML, modern CSS, and lightweight JavaScript.

## Highlights

- Responsive one-page portfolio
- Dedicated credentials library
- AWS badge gallery with Credly verification links
- Downloadable resume
- Live Amazon S3 project demo
- GitHub project links
- No frameworks or build tools required
- GitHub Pages compatible

## Local Preview

Open `index.html` directly in a browser, or use VS Code Live Server.

## GitHub Pages Deployment

1. Create a public GitHub repository.
2. Upload all files while preserving the folder structure.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/root`.
6. Save and wait for the published URL.

## Portfolio Updates

When new project repositories are completed, replace the “Publishing next” text with direct GitHub links.

## Privacy

The public portfolio intentionally excludes phone number and private account identifiers.
