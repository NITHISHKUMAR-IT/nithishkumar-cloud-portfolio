# Deployment Checklist

## Before Publishing

- Confirm all links open correctly.
- Confirm the resume PDF opens.
- Confirm badge images load.
- Confirm credential PDFs open.
- Confirm the S3 live demo is still active.
- Test mobile layout.
- Do not upload AWS account IDs, access keys, passwords, MFA details, or private documents.
